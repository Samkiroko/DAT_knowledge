Creating A course

From you dashboard Navigate to Course on the menu side bar.No the Top left corner Add course.

Steps to follow:

  1. Categories: This as the main course categories set by admin where the course will fall under.
  2. Name: Name of the course the will fall under the main course category. NOTE: Make it unique.
  3. Slug: The name that will show on search url, can be the same or different.
  4. Description:Short paragraph about the course.
  5. Code:unique Number of alphabets to a identify the course form other courses you have created
  6. Image:Small size image to identify the course on the main website, make sure its look amazingüëçüèΩ.
  7. Save and go back to list.

Adding tutoring course

Navigate to Manage ==> courses.Under the tab this is where you will add full details of the course plus the cost.Top left corner click Add tutoring courses

Steps to add a course:

  1. Course Name: Select the Course you create on the Course tab
  2. Duration:How long will the course take choose duration type option
    • Hours
    • Days
    • Months
    • Years
  3. Duration value:The length of the duration value.
  4. ContentTable of content of the course in details.
  5. Time slotsdifferent booking when the course will be available.
    Note: Format is in 24hr system e.g 7-8, 13-14, 15-17, separate the booking with (,).
  6. Days off:Add the days you are not available.
  7. Sort order:Course number.
  8. Save and go back To List

Confirm the course is published

To make sure the course is visible on the main site for student, Logout and navigate to Book a teacher Search yourself click book Now choose course if its available you are good to go.
